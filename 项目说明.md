# 选择算法100讲 - 交互式决策工具平台

## 🎯 项目概述

本项目是一个基于Web的交互式决策学习平台，专为管理者和决策者设计。平台包含三个核心模块：**选择算法100讲教程**、**交互式决策工具导航器**和**用户认证系统**，现已完全适配Cloudflare Pages部署。

## 🏗️ 项目结构

```
bigchoice/
├── index.html              # 选择算法100讲教程（主页面）
├── decision-tools.html     # 深色主题决策工具导航器
├── login.html             # 用户登录验证系统
├── package.json           # Cloudflare Pages配置
└── 项目说明.md           # 本文档
```

## 🚀 部署指南

### Cloudflare Pages部署（推荐）
```bash
# 方法1：直接拖拽部署
1. 打开 https://pages.cloudflare.com
2. 连接GitHub仓库
3. 选择bigchoice文件夹
4. 部署设置：
   - 构建命令：留空
   - 发布目录：/
   - 根目录：/

# 方法2：Wrangler CLI部署
npm install -g wrangler
wrangler login
wrangler pages deploy ./bigchoice --project-name=bigchoice
```

### 本地测试
```bash
# 使用Node.js本地服务器
cd bigchoice
npm run dev
# 访问 http://localhost:3000
```
