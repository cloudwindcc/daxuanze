# 交互式决策工具导航器项目说明

## 项目概述

本项目是一个基于Web的交互式决策工具导航系统，旨在帮助管理者和决策者探索、理解并选择最适合的决策方法。项目包含两个核心页面：

1. **决策工具导航器** (`index.html`) - 交互式决策工具探索平台
2. **选择算法100讲** (`juece.html`) - 系统化选择方法论教程

## 技术架构

### 前端技术栈
- **HTML5** + **CSS3** - 结构和样式
- **Tailwind CSS** - 响应式UI框架
- **Chart.js** - 数据可视化图表
- **原生JavaScript** - 交互逻辑和AI集成
- **Google Fonts** - 字体优化（Noto Sans SC + Inter）

### 核心功能特性

#### 1. 决策工具导航器 (index.html)

**功能模块：**
- **智能推荐系统** - 基于用户问题描述，AI推荐最适合的决策工具
- **分类过滤** - 按类别、关键词筛选21种决策工具
- **可视化图表** - 展示工具类别分布
- **详细工具卡片** - 每个工具包含完整信息：
  - 基本定义和核心内涵
  - 3个实际应用场景
  - 优缺点分析
  - 深度解读
  - 可视化图表（SWOT矩阵、艾森豪威尔矩阵、鱼骨图）

**AI集成特性：**
- **工具推荐引擎** - 基于Gemini API的智能推荐
- **场景案例生成** - 根据用户输入上下文生成定制化应用案例

**决策工具库：**
包含21种经典决策工具：
- SWOT分析、决策树、成本效益分析(CBA)
- 头脑风暴、德尔菲法、帕累托分析
- 六顶思考帽、AHP层次分析法、加权决策矩阵
- 波特五力分析、PESTLE分析、BCG矩阵
- 艾森豪威尔矩阵、5 Whys方法、鱼骨图
- 场力分析、情景规划、风险管理矩阵
- 博弈论、名义小组技术(NGT)、优缺点列表

#### 2. 选择算法100讲 (juece.html)

**内容结构：**
- **8大主题模块** - 系统化的选择方法论
- **100个核心算法** - 从理论到实践的完整覆盖
- **交互式目录** - 粘性导航栏，支持平滑滚动
- **粒子背景动画** - 科技感视觉效果

**内容主题：**
1. **选择基础3讲** - 理性选择、二阶理性、有限道德
2. **选择算法4讲** - 胜率赔率、期望值、凯利公式、贝叶斯主义
3. **财富选择14讲** - 复利、财务自由、价值投资、机会成本
4. **成事选择33讲** - 熵增驾驭、关键时刻、思维格栅、漂亮撤退
5. **理财选择5讲** - 格雷厄姆、巴菲特、查尔斯·埃利斯投资哲学
6. **成长选择25讲** - 人生道路、中年突破、个人含金量、惯性陷阱
7. **生活选择15讲** - 社交构建、消费选择、婚姻决策、移居策略
8. **成为选择高手6讲** - 五种资源、敢于说不、平常心、二阶智慧

## 用户体验设计

### 视觉设计
- **温暖中性色调** - 石色背景配合翡翠绿强调色
- **响应式布局** - 支持桌面端和移动端
- **卡片式交互** - 悬停效果、平滑过渡动画
- **无障碍设计** - 键盘导航、语义化HTML

### 交互流程
```
用户进入 → 浏览工具分类 → 选择/搜索工具 → 查看详情 → 获取AI推荐 → 生成场景案例
```
