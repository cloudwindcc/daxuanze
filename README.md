# 选择算法100讲 - 交互式学习平台

一个现代化的交互式选择算法学习平台，集成了决策工具导航、微信扫码登录、详细教程和实战案例。

## 🚀 项目特性

### 🎯 核心功能
- **交互式决策工具导航器** - 21种专业决策工具的完整解析
- **微信扫码登录** - 支持微信快捷登录和传统账号登录
- **分级学习系统** - 从基础到专家的6个完整学习章节
- **AI智能推荐** - 基于问题描述的智能工具推荐
- **响应式设计** - 完美适配桌面端和移动端

### 📊 决策工具库
- **21种专业工具** - SWOT分析、决策树、成本效益分析等
- **智能筛选** - 按类别、关键词快速筛选
- **详细解析** - 每个工具的定义、应用场景、优缺点
- **可视化分析** - 工具类别分布图表

### 🔐 用户系统
- **微信扫码登录** - 一键快捷登录，自动创建账号
- **分级权限** - 登录后解锁高级学习内容
- **学习进度** - 实时跟踪学习进度和完成情况
- **多端同步** - 支持会话存储和本地存储

## 🎨 视觉设计

### 🌙 现代UI设计
- **深色主题** - 优雅的深色界面，护眼舒适
- **玻璃形态** - 现代化的毛玻璃效果
- **渐变色彩** - 流畅的渐变过渡动画
- **微交互** - 细腻的悬停和点击反馈

### 📱 响应式布局
- **自适应设计** - 完美适配各种屏幕尺寸

## 🎮 使用指南

### 快速开始
1. **访问主页** - 打开 `index.html`
2. **选择学习路径** - 浏览100讲内容或决策工具
3. **登录系统** - 使用微信扫码或账号密码登录
4. **解锁内容** - 登录后解锁高级章节

- **微信扫码登录** - 点击"微信扫码登录"，3秒后自动完成
